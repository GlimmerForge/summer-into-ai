# Week 4 Theme: Built on Yesterday

**Theme:** Built on Yesterday — The Last Week  
**Source:** https://advisoryhour.substack.com/p/summer-into-ai-week-4-the-last-week  
**Dates:** July 6–13, 2026

## Summary

"Tomorrow is built on what came before — so, pick a project you already submitted, or pick someone else's, and build on that."

This is the final week of Summer into AI 2026. The theme honors iterative improvement: take an existing demo (from any week, from any participant) and make it meaningfully better.

## Build Ideas (from theme post)

- Select a previous submission from Summer into AI (50+ projects available) or Spring into AI
- Enhance and improve an existing project using current AI capabilities
- Show visible evolution — the new version should be demonstrably more capable

## Key principle

"We can all enjoy a summer of tinkering with AI because of yesterday." The brief emphasizes the American innovation tradition: honor what came before, then build past it.

## Notes for demos

- Every demo MUST clearly state what it builds on (demo name + week)
- The improvement should be substantial, not cosmetic
- AI integration should be deeper/different than the original
