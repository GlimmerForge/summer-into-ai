# Liberty 1776

**Week 1 · Theme: 8-Bit America · Summer Into AI 2026**

A Space Invaders-style arcade game set during the American Revolution. Fight through 4 waves of enemies based on real grievances from the Declaration of Independence, then face King George III as the final boss. Pixel sprites, 8-bit sound effects, a waving flag, and fireworks on victory.

## Waves

| Round | Grievance |
|-------|-----------|
| 1 | Taxation Without Representation |
| 2 | Quartering of Armed Troops |
| 3 | Cutting Off Our Trade |
| 4 | Dissolving Our Legislatures |
| Final | King George III |

## Controls

| Input | Action |
|-------|--------|
| ← → / A D | Move |
| Space / Z | Fire |
| On-screen buttons | Mobile controls |

## Deploy to Vercel

1. Import `GlimmerForge/summer-into-ai` in Vercel
2. Set **Root Directory** to `projects/week-01-8bit-america/demo-02-liberty-1776`
3. Deploy — no API key needed, this game has no AI calls
