# Week 3 Theme: Data.Punk

> "It's punk because you know the government never built apps for this data."

## Core Concept

Channel punk's spirit of rebellion by transforming boring, indecipherable federal datasets into novel public applications. The government collected it. You make it useful.

**Primary source:** [data.gov](https://data.gov/) — federal datasets across every domain (health, environment, crime, infrastructure, agriculture, economics, weather, education, transportation, etc.)

**Alternative:** Own datasets are acceptable but federal data is preferred for theme alignment.

## Philosophy

- Volume over perfection — ship multiple projects rather than perfecting one
- Civic utility meets creative defiance
- "Can I build a—?" → Yes. Ship it immediately.
- Unlock "civilizational level" value from data nobody has turned into a product yet

## Build Ideas (obvious/expected — avoid or radically subvert)

These are the interpretations every competitor will attempt. Know them so you can go further:

- Simple dataset explorer / search UI
- Chart dashboard for a single dataset
- "Chat with your data" wrapper
- Map visualization of geographic data
- Basic API that exposes a dataset as JSON

## What Strong Demos Look Like

- AI that reasons *across multiple datasets* and surfaces non-obvious connections
- Real-time alerts or monitoring built on live government data feeds
- Personalized tools: "what does this dataset mean *for me* specifically?"
- Adversarial or surprising angles: using government data to expose or critique something
- Games or simulations driven by real federal data (not made-up numbers)
- Tools that answer questions the government never thought to answer with their own data

## Data.gov Starting Points

Explore by category at data.gov. High-signal domains:

- **Health:** CDC datasets, hospital quality, drug adverse events (FDA), food recalls
- **Environment:** EPA air quality, water quality, climate normals, wildfire perimeters
- **Crime:** FBI crime data, NYPD complaints, federal prison stats
- **Economics:** BLS unemployment/wages, Census income, SBA loan data
- **Transportation:** FAA near-misses, NTSB accidents, DOT road safety
- **Agriculture:** USDA crop yields, food deserts, pesticide use
- **Education:** NCES school performance, student loan defaults, Pell grant data
- **Infrastructure:** Bridge conditions, broadband coverage, power outages
