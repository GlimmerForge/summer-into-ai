# Summer Into AI — Competition Rules

> "Small shipped work beats ambitious unfinished work."
> **Build → Publish → Submit → Score → Repeat**

## How to Sign Up

Subscribe to [@advisoryhour on Substack](https://advisoryhour.substack.com). Watch for emails with **"SUMMER INTO AI"** in the subject line — that's how theme drops are announced each week.

---

## What Counts as a Valid Submission

Every scored entry needs **all three** of the following:

| # | Component | What qualifies |
|---|-----------|---------------|
| 1 | **Build Artifact** | App, tool, demo, workflow, prototype, article, video, etc. |
| 2 | **Public Explanation** | Post on LinkedIn, YouTube, blog, or Substack |
| 3 | **Working Proof** | Public demo URL **and/or** shared code repo |

Acceptable proof formats: hosted app, GitHub repo, Hugging Face Space, screen recording, downloadable executable.

> **No points** for complexity claims, effort alone, private work, or unshipped ideas.

---

## How to Submit

Write a Substack post tagging **@advisoryhour** and include:

- Project title and description
- How it connects to the week's theme
- Links to your public posts (LinkedIn, YouTube, blog, Substack, etc.)
- Proof links (demo URL and/or code repo)
- Competitor reference link (if claiming the bonus)

---

## Scoring Formula

```
Total Points = Base Publishing Points + Proof Points + Bonus Points
```

### Base Publishing Points
One entry per platform (Substack, LinkedIn, YouTube, blog, etc.).

- **YouTube** posts must be at minimum 5 minutes, include a clear walkthrough, and identify you as a participant.

### Proof Points

| Proof Submitted | Points |
|----------------|--------|
| Demo only | ✅ Valid |
| Code only | ✅ Valid |
| Demo + Code | ✅ Extra points |
| Neither | ❌ Not scored |

### Bonus Points

| Bonus | How to Earn |
|-------|------------|
| Theme Alignment | Explain the connection in your submission |
| Competitor Reference | Cite or respond to another competitor's work (max 1 per submission) |
| First Submission | Be the first complete submission of the week |

---

## Weekly Schedule

- Theme drops every **Monday** (starting June 15, 2026)
- Submissions due before the weekly deadline announced with each theme
- Late submissions may be showcased but only count toward scoring if Eric grants a grace period

### Multiple Submissions

You can submit more than once per week — each entry needs its own distinct artifact, explanation, and proof.

---

## Winning

Highest total points wins. Tiebreakers (in order):

1. Most weeks with a valid submission
2. Most public demos
3. Most competitor references
4. Public random draw if still tied

**Prizes:** Top 3 finishers receive recognition + medals (details TBA).

## The Slacker Board

Any participant with zero submissions in a given week gets humorously listed on the **Slacker Board** and owes the group a coffee. Don't be on the Slacker Board.
